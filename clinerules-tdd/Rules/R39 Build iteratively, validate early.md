# 💡 Build iteratively, validate early

> Build iteratively: implement the minimum viable solution first, then evolve. Validate early: test assumptions with working code as quickly as possible. Question assumptions. Defer irreversible decisions until you have maximum information.

**Type:** PREFERENCE  ·  **Hand tag:** PREFERENCE  ·  **LLM tag:** PRESCRIPTION

## Judgment
Consolidates four philosophy bullets. Annotator flags 'defer until maximum information' as potentially infinite regress — you never provably have maximum information. Philosophy, not enforceable norms.
